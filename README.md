## 基础命令

```
settime 2 5 0 \\ 设置为第二天早上五点
teleport -558 1042 \\ 传送到 558W 1042N
giveself ammo9mmBulletBall 6 1 \\ 9毫米子弹包 
giveselfxp 500  \\ 加经验
```
